# CRUD Products - DevOps Project

Este proyecto implementa un CRUD de productos en Python utilizando archivos JSON como almacenamiento.

## Funcionalidades

- Crear productos
- Leer productos
- Actualizar productos
- Eliminar productos

## Pruebas

Las pruebas se ejecutan con pytest.

## Pipeline CI/CD

El pipeline automatiza:

- Instalación de dependencias
- Verificación de estilo con flake8
- Ejecución de pruebas unitarias

El pipeline se ejecuta automáticamente en GitHub Actions cuando se realiza un push al repositorio.
